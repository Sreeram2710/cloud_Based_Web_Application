# Cloud Security Solutions Website

A single-page responsive website showcasing cloud security solutions, focusing on AWS CI/CD pipeline implementation and antimalware protection for virtual machines.

## Overview

This static website highlights:
- AWS Cloud hosting with CI/CD pipeline integration
- Antimalware solutions for virtual machines
- Modern, responsive design

## Features

- **Responsive Design**: Optimized for all devices (mobile, tablet, desktop)
- **Modern UI**: Clean layout with card-based sections
- **Easy Navigation**: Simple single-page structure
- **Footer**: Copyright information for Sreeram

## Deployment

This website is designed to be hosted on AWS using a CI/CD pipeline:

1. **Source Control**: Store code in a Git repository (GitHub, GitLab, AWS CodeCommit)
2. **CI/CD Pipeline**: Set up using AWS CodePipeline or similar service
3. **Hosting**: Deploy to AWS S3 or AWS Amplify for static hosting
4. **CDN**: Optionally configure Amazon CloudFront for global delivery

## Local Development

To run this website locally:
1. Clone the repository
2. Open `index.html` in your browser
3. Make changes using any text editor or IDE

## Technology Stack

- HTML5
- CSS3 (with responsive design principles)
- No JavaScript dependencies
